# OOP Assignment

A simple demonstration of Object Oriented Programming principles using PHP.

## About

This project demonstrates how object oriented programming works in PHP by creating a hierarchy of instrument classes.  
Different types of instruments inherit behaviour from a parent class and override methods where needed.

### Language

PHP

### Tools

Git
Vscode

## Features

- Valid PHP
- Object oriented programming structure
- Parent class with multiple child classes
- Use of an interface
- Use of a trait
- Method overriding
- Autoloading with `spl_autoload_register`
